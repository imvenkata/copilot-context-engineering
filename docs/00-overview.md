# Overview

This pack standardizes Copilot usage in Python repos. It provides:
- Repo-wide and path-specific instructions
- Repeatable prompt workflows
- Agents for plan/implement/review/triage/infra
- Skills for tests, refactors, Terraform, CI, and reviews
- GitLab-first workflows for merge requests, issues, epics, and CI

Start with the README, then customize instructions and prompts for your repo.
